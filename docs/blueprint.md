# **App Name**: LoginFlow

## Core Features:

- Email/Password Authentication: Allow users to sign up and sign in using their email addresses and passwords.
- Google Sign-In: Enable users to sign in using their Google accounts.
- Form Toggle: Provide a toggle to switch between login and registration forms.
- Loading animation: Display Instagram's logo with a loading animation when the app starts.
- Professional Homepage: Use an AI tool to recommend an informative and aesthetically pleasing layout for the user's personal home page after login.

## Style Guidelines:

- Primary color: Dark teal (#008080) for a modern, trustworthy feel.
- Background color: Light gray (#ECEFF1) for a minimalist aesthetic.
- Accent color: Teal (#00BFA5) for highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif font, will provide a modern, neutral look for both body text and headlines.
- Smooth transitions between the login and registration forms.
- Modern and minimalist layout with a focus on user experience.
- Use clean, simple icons for form fields and the Google Sign-In button.
